#### 10. Many values to multiple variables
```
x, y, z = "Orange", "Banana", "Cherry"
print(x) - Orange
print(y) - Banana
print(z) - Cherry
```

#### 11. One value to multiple variables
```
x = y = z = "Orange"
print(x) - Orange
print(y) - Orange
print(z) - Orange
```

#### 12. Unpack a collection
```
fruits = ["apple", "banana", "cherry"]
x, y, z = fruits
print(x) - apple
print(y) - banana
print(z) - cherry
```

#### 14. Global variables

**Global variables:**
	x = "awesome"
	def myFunc():
		print("Python is " + x)
	myFunc()

**Global keyword:**
	def myFunc():
		global x
		x = "fantastic"
