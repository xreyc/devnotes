The React `useCallback` Hook returns a memoized callback function.

The `useCallback` and `useMemo` Hooks are similar. The main difference is that `useMemo` returns a memoized _value_ and `useCallback` returns a memoized _function_. You can learn more about useMemo in the useMemo.

#### Problem
In this scenario, we want to memorize a specific component so that it would only render if one of its props updates. In this case we have a **todos** and **addTodo** prop.

When we increment the count state, we can see that the **Todos** components still rerender, main reason is that the **addTodo** function get recreated everytime. State of the useState can be retained and tracked but normal function gets recreated because they are not a state.

index.js
```jsx
import { useState } from "react";
import ReactDOM from "react-dom/client";
import Todos from "./Todos";

const App = () => {
  const [count, setCount] = useState(0);
  const [todos, setTodos] = useState([]);

  const increment = () => {
    setCount((c) => c + 1);
  };
  const addTodo = () => {
    setTodos((t) => [...t, "New Todo"]);
  };

  return (
    <>
      <Todos todos={todos} addTodo={addTodo} />
      <hr />
      <div>
        Count: {count}
        <button onClick={increment}>+</button>
      </div>
    </>
  );
};

const root = ReactDOM.createRoot(document.getElementById('root'));
root.render(<App />);
```

Todos.js
```jsx
import { memo } from "react";

const Todos = ({ todos, addTodo }) => {
  console.log("child render");
  return (
    <>
      <h2>My Todos</h2>
      {todos.map((todo, index) => {
        return <p key={index}>{todo}</p>;
      })}
      <button onClick={addTodo}>Add Todo</button>
    </>
  );
};

export default memo(Todos);
```

#### Solution
on this case, we wrap the function with a useCallback which updates only the function if one of its dependencies changes. This way the function we get recreated unless the state of the useCallback dependency get updated.

index.js
```jsx
import { useState, useCallback } from "react";
import ReactDOM from "react-dom/client";
import Todos from "./Todos";

const App = () => {
  const [count, setCount] = useState(0);
  const [todos, setTodos] = useState([]);

  const increment = () => {
    setCount((c) => c + 1);
  };
  const addTodo = useCallback(() => {
    setTodos((t) => [...t, "New Todo"]);
  }, [todos]);

  return (
    <>
      <Todos todos={todos} addTodo={addTodo} />
      <hr />
      <div>
        Count: {count}
        <button onClick={increment}>+</button>
      </div>
    </>
  );
};

const root = ReactDOM.createRoot(document.getElementById('root'));
root.render(<App />);
```
