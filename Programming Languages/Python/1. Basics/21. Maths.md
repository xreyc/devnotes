#### 1. Build-in math function
```
x = min(5, 10, 25)  
y = max(5, 10, 25)  
x = abs(-7.25)
x = pow(4, 3)
```
#### 2. Math module
```
import math
x = math.sqrt(64)
x = math.ceil(1.4)  
y = math.floor(1.4)
x = math.pi
```
